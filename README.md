## Summary
This is a HTML address input form that checks the validity of the fields with javascript/jQuery before submitting the information.

## Form fields
The form asks and checks for the following while typing:

  *  House number
  *  Street
  *  City
  *  State
  *  Zipcode
  *  Unit Number
